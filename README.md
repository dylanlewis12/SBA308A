# Pokémon GO - Search & Deck Builder

A web application for discovering, searching, and collecting Pokémon.

## Features

- **Home Page** - Auto-rotating Pokémon carousel with dynamic type-based card colors
- **Search Page** - Real-time search with suggestions and "Add to Deck" functionality
- **Deck Page** - View and manage your saved Pokémon collection with delete option

## Technologies

- HTML, CSS, JavaScript
- Axios for API requests
- PokéAPI (10,000+ Pokémon)
- LocalStorage for data persistence
- Google Fonts 

## Usage

**Home Page** - Watch Pokémon rotate every second with type-based card colors

**Search Page** - Type 2+ characters to search, click a suggestion, and add to your deck

**Deck Page** - View all saved Pokémon or clear your entire collection
